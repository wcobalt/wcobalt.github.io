Dialog Helper Writer

Our Add-on enables writers to easily generate filenames of dialogue and voiceover assets based on data from other columns of your Linebook Spreadsheet. The Add-on has three modes:

Generate IDs and filenames

When this mode is selected the Add-on updates all the lines of a Spreadsheet. Use this with caution, because it may take a lot of time to regenerate data for the whole spreadsheet.

Generate IDs and filenames (Range)

This mode updates rows only in currently active range.

Generate IDs and filenames (Advanced)

This mode allows to manually specify what rows are to be updated. Use either a range, e.g. 3-10000 to specify range or just a number, e.g. 1000 to specify what number of columns starting from the 3rd one it's needed to update or use empty input to specify that the whole spreadsheet is to be updated.

What does the Add-on do?

For each row Content column of which is empty the following is executed: if the row's LineID is empty then LineID is generated and set for this row. After that (no matter whether LineID was generated or was already set) the dialogue asset filename and the voice over wave sound filename are generated.
